Tutorial

In the interactive Google Maps you can click on the map to move the marker. This marker is the center of the square kilometre area where the road surface are calculation is done. Clicking and holding onto a marker allow you to move the marker around the map. The marker is automatically the center of the map.

This button change the Google Maps view type. It changes the default view type which is 'roadmap' to 'satellite'. In 'satellite' view type, the map shows satellite images instead of graphical images.

Google Maps display the map using satellite images.

This button style the Google Maps. Styling Google Maps allow us to manipulte the objects in the map such as labels, colours of roads and building, and visibility of objects. With this button, the map is styled so that only roads are shown more clearer than all other objects.

Here is an example of a 'road only' style shown on the map. The colour of the roads are black, while everything else is grey.

This button changes the style back from 'road only' to the default style that the map initially uses.

This button takes the current coordinates of the marker and use it to calculate the estimated total surface area of all roads in the square kilometre.

Once the result is calculated it is displayed underneath the map control buttons. The information display shows the total surface area and the time it took to do the calculation. The time it took to do the calculation is timed in the server, where it calculate the elapsed time it took to create the image URL, process the pixels, make the pixels to surface area calculation.
